Tags importantes no HTML

Negrito: Deixa a fonte em negrito, mas os leitores de página não entendem esta enfase.Tag Física

Strong:Assim como o negrito, destacam o texto, mas adicionalmente eles dão enfase ao conteúdo, de forma que leitores de tela conseguem enfatizar o conteúdo.Tag Semântica

Itálico: Destacam o conteúdo, mas sem afetar os leitores de página.Tag Física

Sublinhado: Destacam o conteúdo, mas sem afetar os leitores de página.Tag Física

Sobrescrito>:Sobrescreve o contéudo, mas sem afetar os leitores de páginas. Tag Física

Podemos sobrescrever uma parte do texto indicando que a palavra está "incorreta" e inserir a palavra correta em substituição a primeira.Estas tags influenciam o comportamento dos leitores de páginas.Tag Semântica

Small:Diminuem o tamanho do conteúdo.Colocadas em cadeia diminuem ainda mais...

Potênciação:Usado para colocar o texto pequeno e acima. Exemplo: 22

Pequeno e abaixo: Semelhante a potenciação, mas fica abaixo. Exemplo: H2O

citação: Coloca o conteúdo entre aspas duplas indicando uma sitação, normalmente terá um link para um texto explicativo.

Code:Usado para ter uma fonte específica que representa linhas de código. Usado com a tag pre formata o texto de acordo com o descrito.
Exemplo: 


            a=1
            b=2
            c = a + b

Span:Tag de contexto genérico para texto inline, serve para aplicar estilos diferenciados em parte do texto.

Div:Cria um parágrafo com o conteúdo. Exemplo:
Eu sou um bloco
